¿Qué es la Inteligencia Artificial?

La IA es el campo que busca crear sistemas capaces de percibir, razonar, aprender y actuar. Hoy abarca desde modelos predictivos hasta agentes autónomos que combinan datos, conocimiento y objetivos.

  • Enfoques: simbólico, conexionista y híbrido.
  • Paradigmas: aprendizaje supervisado, no supervisado, por refuerzo.
  • Actualidad: IA generativa, RAG, agentes y edge AI.
Curso ML (Google) MIT OCW

Tipos de IA

ANI (estrecha), AGI (general) y ASI (superinteligente). También se clasifican por técnicas: reglas, redes neuronales, modelos generativos.

Aplicaciones actuales

Salud (diagnóstico), finanzas (detección de fraude), educación (tutores), industria (mantenimiento predictivo) y gobierno (servicios digitales).

Machine Learning: fundamentos

El ML permite que los sistemas aprendan patrones desde datos para realizar predicciones o decisiones con mínima intervención humana. El flujo típico: datos → preparación → entrenamiento → evaluación → despliegue → monitoreo.

  • Conjuntos de datos representativos y balanceados.
  • Características (features) relevantes y limpieza.
  • Métricas: precisión, recall, F1, AUC, MAE/MSE.

Regresión

Modela relaciones numéricas (p.ej., precios). Variantes: lineal, regularizada, árboles.

Clasificación

Asigna etiquetas (spam/no spam). Algoritmos: SVM, árboles, redes.

Clustering

Agrupa sin etiquetas (k-means, DBSCAN) para segmentación y exploración.

Redes Neuronales

Capas de neuronas para visión, lenguaje, audio; base de la IA generativa.

Casos de uso: recomendadores, mantenimiento predictivo, detección de fraude, visión para calidad, asistentes conversacionales.
